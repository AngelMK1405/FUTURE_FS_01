## Setup Steps
Follow these steps to run this project locally on your computer:

### 1. Download or Clone Repository
Download the ZIP file or clone the repository using Git:

```bash
git clone https://github.com/AngelMK1405/FUTURE_FS_01.git
```

OR manually download the project files from GitHub.

### 2. Open in VS Code
Open the project folder in **Visual Studio Code**.

### 3. Run the Project
Open `index.html` in your browser.

You can:
- Double-click `index.html`
OR
- Use VS Code Live Server extension for better development experience

## Deployment Steps
This project was deployed online using **Netlify**.

### Step 1
Completed the website development using HTML, CSS, and JavaScript in VS Code.

### Step 2
Created a public GitHub repository named:

```bash
FUTURE_FS_01
```

### Step 3
Manually uploaded all project files to the GitHub repository.

Files uploaded:
- index.html
- style.css
- script.js
- AMK_resume.pdf
- README.md

### Step 4
Logged into Netlify using GitHub account.

### Step 5
Imported the GitHub repository into Netlify.

### Step 6
Deployed the website and generated a live public URL.
